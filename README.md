# dictionaries

Benchmarks for dictionary data structures: hash tables, maps, tries,
etc.

The `judy` package was removed from this test suite for instability;
it segfaults the program.

## Running

For all benchmarks:

    $ stack bench

For just space:

    $ stack bench :space

For just time:

    $ stack bench :time
